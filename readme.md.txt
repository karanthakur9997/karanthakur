# Happy Birthday My Love

A beautiful and heartfelt birthday slideshow website to express your love, featuring balloon animations and firecracker effects.

## How to Use

- Open `index.html` in any modern browser.
- Click "Next" buttons to cycle through slides.
- On the last slide, click "Start Again" to restart the slideshow.
- Balloon and firecracker animations run continuously in the background.

## Hosting on GitHub Pages

1. Create a new GitHub repository.
2. Upload the `index.html` file to the repository root.
3. In repository settings, enable GitHub Pages (choose the main branch).
4. Visit the provided GitHub Pages URL to see your site live.
